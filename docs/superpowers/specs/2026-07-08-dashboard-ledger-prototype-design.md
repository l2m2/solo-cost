# 仪表盘「账本」风格原型（方案 B · 仅总览 Tab）

日期：2026-07-08
分支：`feat/dashboard-ledger-prototype`

## 目标

给「沃工本」一个有辨识度的视觉身份：把仪表盘改成中式账本气质（宣纸底、
墨黑/朱砂红双色、印章签名）。先从总览 Tab 做原型验证，认可后推广到整个仪表盘。

> 进展：总览验证通过后已扩展到**三个 Tab + 年度明细弹框 + 页头**。印章仍只在总览
> hero 一处（系统统一而非到处盖章）。其它页面不变。

## 设计令牌

- **配色**：宣纸米 `#F7F4EC`、墨黑 `#1C1A17`、墨灰 `#6B6459`、朱砂红 `#B23A2E`、
  靛蓝 `#3B4A6B`、栏线 `#D8CFBF`。
- **字体**：数字与标题用系统中文衬线栈
  `"Songti SC","STSong","Source Han Serif SC","Noto Serif SC","SimSun",serif`
  （桌面系统自带，不打包字体）；标签用现有无衬线。
- **签名元素**：核心「已收到手」数字旁盖一枚朱红印章（篆意"沃"字、双环、微旋转）。
  只此一处，其余极克制。

## 范围与实现

- 新增 `src/components/dashboard/LedgerOverview.tsx`，自包含账本样式，
  样式全部用显式颜色（Tailwind 任意值），**不改** `globals.css` / tailwind 主题，
  避免影响其它 Tab 和页面。
- 结构：
  1. 账本抬头「经营账」+ 双线分隔。
  2. Hero：已收到手大号衬线数字 + 印章；下方一行扣减明细。
  3. 两个口径账块（合同口径 / 已收口径），各三行栏线条目；「到手」行加重，
     「未收/应收」用朱砂红。
  4. 按年到手：栏线进度条（到手=靛蓝、已收=墨灰）+ 图例，点击仍打开年度明细弹框。
- `dashboard.tsx`：总览 Tab 的财务区块替换为 `<LedgerOverview>`；保留其下的
  `TodoTasksCard` 与年度明细 `Dialog`。移除因此不再使用的 `SectionTitle`、`LegendDot`
  两个辅助组件（`Kpi` 仍用于应收 Tab、`Bar` 仍用于分布 Tab，保留）。

## 推广范围（原型通过后）

- 共享 `ledgerParts.tsx`（`LedgerPanel` / `LedgerBar`），各账本"页"统一。
- 待办卡片、分布&排行（状态分布 + 排行榜）、应收表、年度明细弹框、页头全部账本化：
  纸底面板、栏线表格、衬线数字、朱砂红用于支出/逾期、靛蓝用于净额/进度。
- 状态与应收档位用小圆点替代彩色徽章；排行名次前三用朱砂红。

## 范围之外

- 不改其它页面、不改数据与接口。
- 不打包 Web 字体（用系统衬线回退）。

## 验证

- 无逻辑改动；`pnpm tsc`、`oxlint`、`pnpm build` 通过。
- 手动：总览 Tab 账本渲染正常、印章显示、年度点击弹框仍可用；另两个 Tab 不受影响。
- 与 `main` 并排对比后由用户决定去留。
